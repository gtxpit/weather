# Weather App 🌤️

Погода и прогноз на 5 дней по городу или геолокации.

🔗 **[Открыть сайт](https://gtxpit.github.io/weather/)**

## Возможности

- Поиск города
- Геолокация
- Прогноз на 5 дней
- Избранные города
- Фон под погоду

---

## Технологии

HTML, CSS, JavaScript, OpenWeatherMap API, LocalStorage
